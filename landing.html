Features
Everything you need
to reason spatially

From quick sanity checks to rigorous expert-level analysis — Forma adapts to your workflow.

Three reasoning modes
Quick 2-step answers for fast sanity checks. Deep structured analysis for design review. Expert-level technical output with formulas and quantitative estimates.
Image annotation
Draw arrows, rectangles, circles, and freehand marks directly on your image before analysing. Direct the model's attention to exactly what matters.
Streaming output
See reasoning appear token-by-token in real time. No waiting for the full response — follow the model's logic as it unfolds.
Conversation history
Ask follow-up questions with full image context preserved. Build iterative analyses across multiple turns without re-uploading.
Projects
Organise analyses into named collections. Bulk-export an entire project as a polished PDF report — ready for client delivery.
Batch analysis
Upload dozens of images at once and run the same question across all of them. Process an entire photo survey in minutes instead of days.
